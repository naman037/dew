# Design System: Modern Clothing Brand

This design system defines the aesthetic and visual language for generating screens using Stitch MCP.

## 1. Brand Identity
- **Vibe:** Modern, minimal, fashion-forward, premium branding. 
- **Style:** High-end fashion photography style, clean whitespace, strong typography, layout optimized for conversion.
- **Experience:** Mobile-first responsiveness, smooth micro-interactions.

## 2. Platform & Theme
- **Platform:** Web
- **Layout:** Mobile-first, fully responsive
- **Theme:** Light theme, high contrast

## 3. Colors
- **Background:** Pure White (`#ffffff`) for a clean, minimal look.
- **Surface:** Very Light Gray (`#f9fafb`) for subtle sectioning or cards.
- **Primary Accent:** Pure Black (`#000000`) for primary buttons, prominent text, and call-to-actions, giving a luxury feel.
- **Text Primary:** Pure Black (`#000000`) for high-contrast readability.
- **Text Secondary:** Dark Gray (`#4b5563`) for descriptions and metadata.
- **Borders/Dividers:** Light Gray (`#e5e7eb`) for subtle separation.

## 4. Typography
- **Headings:** Strong, bold, sans-serif or elegant serif typography to feel editorial and luxurious.
- **Body:** Clean, legible sans-serif. Use generous line height (1.5x) for breathability.
- **Whitespace:** Abundant whitespace around elements to give the design "breathing room" typical of high-end brands.

## 5. UI Elements
- **Buttons:** Sharp or very subtly rounded corners. Background black (`#000000`), text white (`#ffffff`). Hover state: Dark gray (`#1f2937`) or subtle outline inverted.
- **Cards/Images:** Edge-to-edge images or clean grid layouts with no borders. High-fidelity product photography.
- **Navigation:** Understated, persistent, and clean. Usually transparent or solid white background.
- **Micro-interactions:** Smooth fade-ins, subtle image scaling on hover, clean drawer/cart slide-outs.

## 6. Design System Notes for Stitch Generation
**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Mobile-first
- Theme: Light, minimal, modern, premium fashion
- Background: Pure White (#ffffff)
- Surface: Very Light Gray (#f9fafb)
- Primary Accent: Pure Black (#000000) for buttons and CTAs
- Text Primary: Pure Black (#000000)
- Text Secondary: Dark Gray (#4b5563)
- Style: Clean whitespace, strong typography, high-end photography layout
- UI Elements: Sharp buttons, edge-to-edge imagery, subtle micro-interactions
